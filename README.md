> 为了进一步实现整洁架构，使用 mvp 的方式组织你的前端代码，让项目更加清晰

- Model，一般是要在界面上显示需要的数据，或者临时数据

- View，一般是 react，vue 之类的视图层，它显示数据，并将事件绑定到 Presenter

- Presenter, 提供方法和 Model 给到 View
